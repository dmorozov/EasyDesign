**Badge** — small status pill for save state, frame mode, counts, or feature markers.

```jsx
<Badge tone="success" dot>Saved</Badge>
<Badge tone="warning" dot>Saving…</Badge>
<Badge tone="accent">Web</Badge>
<Badge tone="outline">Email</Badge>
```

Tones: `neutral`, `accent`, `success`, `warning`, `danger`, `outline`. `dot` adds a leading status dot in the current color.
